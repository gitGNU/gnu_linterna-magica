The purpose of this page is to provide a way for download statistic, because statistics for JavaScript files are not handled well by most web server log analyzing software .

For GNU IceCat, Abrowser/Web Browser and other free software versions of Firefox, with installed Greasemonkey extension, download and installation should start automatically after 10 seconds.

Please use the link on the right to download the userscript, if the download process does not start automatically. Sorry for the inconvenience.
